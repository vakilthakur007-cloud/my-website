# Emplaza Creations Website

Welcome to the official repository for the **Emplaza Creations** website. This project showcases our exquisite Maithili embroidery, graphic design services, and digital marketing expertise.

## Project Overview

Emplaza Creations bridges the gap between traditional Maithili art and modern digital innovation. This website serves as a digital catalog for our products and a portfolio for our professional services.

### Features

-   **Maithili Aesthetic**: A unique design inspired by the vibrant colors and geometric patterns of Mithila art (Madhubani).
-   **Service Portfolio**: detailed sections for Graphic Design, Digital Marketing, and Embroidery Design.
-   **Product Catalog**: A showcase of over 30 handcrafted products including paintings, cushions, and decor.
-   **WhatsApp Integration**: Direct "Order on WhatsApp" functionality for seamless customer communication.
-   **Responsive Design**: Fully optimized for mobile, tablet, and desktop devices.

## Setup & Deployment

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/vakilthakur007-cloud/my-website.git
    ```
